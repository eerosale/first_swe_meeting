The Problem

Circular spray pattern is ineffecient

-Food residue is left in corner dishes

-Cleanliness is dependent on dish orientation

Our Solution

We made an improved dishwasher with the following criteria:

  • Improves area coverage in dishwasher

    • Replaced rotating arm with novel mechanical system

  • Hits more angles than a normal dishwasher

    Linear motion rack has two rotating heads

    • The Mechanical Design

    Water pressure spins jets and Cleans Dishes

    (continued)

    Motor 180 rpm (A)

    3:1 Gear reduction to sheave (B)

    12:1 power screw (D)

    Net movement 5"/min

    The Electrical Design

    How we made the system autonomous

  • 12V Gearhead Motor
  • Three relays to switch motor direction
  • Waterproof limit switches placed
  • Arduino Microcontroller used

    • The Computer Science Design

    How we made the project user friendly

  • Arduino serially communicates with Python User Interface
  • Two options for length of Cycle
  • Allows for future work

    • Results

    Dishwasher Improvements Achieved the Stated Design Goals:

    • Jets are aimed tangentially to spinner to cause rotation

    • Additional jets provide washing action

    • More coverage is seen through acrylic casing

    Future Work

    Project leaves great room for improvement

    • Use better material to prevent rust
    • Allow a reset position in case of black-out
    • Have one power source for whole system
    • Use a different microcontroller
    • Make it wifi enabled and have a Web App (cool!)
